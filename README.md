## Run Locally

Clone the project

```bash
git clone https://github.com/96ers/summerizIT.git
```

Go to the project directory

```bash
cd summerizIT
```

Create environment by conda

```bash
conda env create -f environment.yml
```

Active conda environment

```bash
conda activate venv
```
